第一章

代码的最终运行

编译:首先编译器将源代码编译为目标代码。
链接:链接程序将目标代码和C++库函数的代码以及一些标准的启动代码组合起来生成最终的可执行代码。

第二章

基础

main函数

与Java类似,C++也需要一个main函数做入口,这个函数会被启动代码调用,与Java不同的是他的返回值和参数好像并没有强制要求(警告而非错误)。一些动态链接库可以没有main函数。

预处理器

预处理器主要用于对源文件进行编译前的处理。

#include

该编译指令导致预处理器将某些文件(有点像是jar包?)的内容添加到程序中,随后文件内容和源代码一起被交给编译器。

头文件

包含文件或头文件支持一系列实现好的特定的工具,可以通过include指令包含在其他程序中,C++的头文件没有扩展名,其可以包含名称空间。

名称空间

很像Java中的包名,# include 后还需通过using namespace std引入名称空间使得iostream中的定义可用,using namespace可以位于某个函数内(只在该函数内生效),也可以位于函数外(当前程序内生效),也可以通过std::xxx引用名称空间的具体的定义。

C++语句

cout/cin

这两个是定义在iostream中的对象,是对于IO流的一种抽象。
cout << 和 cin >>可以根据要输出/输入的变量的类型自动进行转换。eg:int类型的打印时会将整数转换为字符串,cin >> 也会根据要操作的变量自动转换键盘输入的类型。

函数

C++要求为程序中使用的所有函数提供原型(使用头文件内的函数则不需要),以分号结尾并放在main函数前面,函数同Java一样,支持重载。

第三章

简单变量

变量名称

以单双下划线开头的变量名称是保留名称(什么意思?),不建议使用,与Java不同,C++似乎更常用下划线分割单词而不是用驼峰标识。

整形

C++没有对short/int/long/long long的长度进行硬性规定,而只是规定short至少16位,int至少与short一样,long至少32位且至少与int一样,long long至少64位且至少与long一样(本机测试长度分别为16,32,32,64)。

初始化

C++变量的初始化不同于Java,如果没有初始化,变量的值就是该变量所在内存中的旧值,无法预测,而不会像Java那样进行诸如0的自动初始化。
C++可以使用大括号来初始化任何类型,eg: int a {1},如果使用大括号而没有指定值,那么默认初始化为0;

值的溢出

对于整形的溢出,他会回到另一端的极值。

目前我奇怪的是,对于A处的输出方式,最终输出的是32678,即超过了short的界限,猜测是cout自动提升了类型……

整形字面值

C++可以接受8/10/16进制,分别以1-9/0/0x开头,可以使用cout << oct/hex使得cout能够将一个数字以8/16进制输出。

常量类型确认

对于整形除非超过范围,否则默认为int。

char/wcha_t

char类型在内存中以数字存储,其可以进行+-操作,cin/cout在进行操作时会自动转换。
char只有一个字节无法表示汉字,wcha_t两个字节可以用来表示汉字,由于cin/cout将输入输出看做char流,所以无法处理wcha_t,iostream提供了wcin/wcout。
在赋值时,要通过L来指示宽字符常量。

感觉在输出汉字这方面,C++要比Java麻烦不少,Java的char默认采用的就是Unicode,输出方面也不需要setlocale(这个设置目前还不知道在设置什么),另外吐槽一下,vs2017对于“UTF-8“的支持太不上心了吧……

const

在声明变量方面,有点类似于Java的final,一旦初始化,不允许更改,谷歌风格指南中建议const变量名以k开头。

浮点数

浮点类型

C++有float/double/long double三种浮点类型,对于浮点数,默认以double存储,可以通过f/F/l/L制定存储方式。

算术运算符

类型转换

C++支持函数传参时的自动类型转化,

它会产生警告但不报错。对于强制类型转化,C++提供一种函数调用形式的转化:”typename (value)”,这种设计感觉逻辑上更加清楚,好于Java。

auto声明

auto是一个关键字,他可以根据初始值的类型推断变量的类型,在处理复杂类型时很有用。

第四章

数组

初始化

数组初始化时可以只初始化部分元素,编译器将其他元素设置为0,如果大括号初始化时不包含任何元素所有的元素都会被置为0。

字符串

C风格字符串

C风格字符串必须以空字符‘\0‘结尾,cout输出字符串时直到遇见’\0‘才停止,另外C++的输入工具读取键盘输入到char数组时会自动加上空字符,双引号内的字符默认以空字符结尾,“S”表示两个字符:’S‘和’\0‘。

字符串拼接

拼接字符串时,第二个字符串的第一个字符自动取代第二个字符串的空字符。

字符串输入

cin使用空格/制表符/换行符来确定输入字符串的结束位置。
cin提供了getline/get两个函数用于读取一行输入直到换行符并自动补全空字符,不同的是getline读取后会将换行符丢弃,get则会将换行符留在输入流中,由于这个性质,使用get进行两次读取间需要插入一个get用于读取换行符。
cin.getling()/get()都会返回一个cin对象。

string类

string类在头文件string中,可以使用访问数组的方式访问string对象中的字符。

赋值/拼接/附加

可以将一个string对象赋值给另一个string对象,而且属于值复制(深拷贝?)。

原始字符串raw

原始字符串中字符就表示本身,\n不再表示换行,其使用“(和)”作为定界符,并使用前缀R标识,另外语法允许在“和(之间加入其它字符来重新设置定界符。

结构

创建结构首相要定义结构描述,然后创建结构变量。

定义结构描述可以放在main前面,所有函数都可以使用,也可以定义在函数内。

共用体

共用体是一种数据格式,它能够存储不同的数据类型,但任一时间内它只能存储其中的一种类型。

共用体主要是为了应对”数据量唯一,格式不唯一“的情况,同时可以节省存储空间。同时共用体可以声明在结构体内部并不需要名称(匿名共用体)。

枚举

虽然枚举的符号常量对应着整数值,但是不能对其进行+-运算,或者说进行+-运算时枚举量被提升为int,同时枚举量的值可以使用赋值运算符显式指定。

指针和自由存储空间

地址运算符&可以获得一个变量的地址,*可以获得一个指针变量存储的地址值所指向的值(……)

new分配内存/delete释放内存

不用delete同一个内存块两次,delete只能用来释放new分配的内存。

new创建数组

在操作数组时,指针和数组名等效,psome[1]指向的就是第二个元素,指针可以进行+-运算,psome = psome+1;表示的psome指向的是第二个元素的地址。

指针,数组和指针算术

test和&test的值都是一样的,但数组名被解释为第一个元素的地址,所以对其+1表示的是下一个元素的地址,&test被解释为整个数组的地址,对其+1表示的是下一个这么大的数组的地址。

指针和字符串

如果给cout提供一个字符的地址,他将从该字符起打印直到遇到空字符。
C++中用双引号括起来的字符串像数组一样,也是第一个元素的地址。

自动存储,静态存储,动态存储

在函数内部定义的常规变量使用自动存储,称其为自动变量(栈变量),在函数被调用时,自动生,函数结束时消亡,有丶像Java中的局部变量。
静态存储是整个程序执行期间都存在的存储方式可以将其声明在函数外面,也可以使用static关键字。
C++内部管理了一个内储存池,称之为自由存储空间或堆,new/delete操作内存就是内存池中的内存,如果不delete,即使由于作用域等原因导致指针包含指针的内存被释放,但内存池中的内存也不会被释放。

数组的替代品

vector

vector对象在插入或添加值时可以自动调整长度。

array

array对象和数组对象分配在栈上,vector分配在堆上,array对象也可以进行赋值(值拷贝),同时C++并不检查数组的下标越界问题。

第五章

for循环

复合语句

如果在一个语句块中声明一个变量,外部语句块也有这个名称的变量,那么从声明位置到内部语句块结束的范围之内,新变量会隐藏旧变量。

逗号运算符

逗号运算符确保先计算第一个表达式然后计算第二个表达式(顺序点),逗号表达式的值是第二部分的值。

字符串的比较

对于char数组形式的字符串之间的比较如果使用==比较的是地址,应该使用如果希望比较内容,应该使用strcmp函数(Java的compare方法)。
string对象的比较是在比较内容而不是地址,它使用了重载操作符(!=/=\=),他要求比较的两个对象中至少有一个为string对象。

IO

文件尾条件

当检测到EOF后,cin将eofbit和failbit都置为1,可以通过eof()/fail()函数进行检测。
当cin检测测到EOF后,cin将不再读取输入,此时需要调用cin.clear()清除EOF标记。istream类提供了可以将istream对象(eg:cin)转换为bool值的方法,如果最后一次读取成功了,返回true,反之。
